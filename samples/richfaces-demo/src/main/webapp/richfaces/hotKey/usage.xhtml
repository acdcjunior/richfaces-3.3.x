The hotKey component allows registering hot keys on the page or particular elements and defining client side processing functions for these keys.

Main component attributes are:

  • key - hot key definition ("up", "ctrl+a", etc...)
  • handler - defines callback function for this hotKey
  • selector - JQuerry selector, defines elements which will process defined hotKey(html[0] by default)

First example shows you simple hotKey components usage. In order to enlarge or reduce image size - press CTRL + Up/Down combinations. and in order to browse between images - use CTRL + left/right keys.
