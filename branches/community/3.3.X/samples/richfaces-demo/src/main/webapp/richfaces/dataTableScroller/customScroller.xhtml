This sample is designed to show how to use datascroller component facets for component look-and-feel and functionality customization.

As you could see the datascroller customizes step controls and default buttons changes to select which allows to switch to the four nearest pages to the current one.

Data Scroller customization

The next new JS API functions are introduced and were used to build this sample:

  • switchToPage(pageToGoValue)
  • next()
  • previous()
  • first()
  • last()
  • fastForward()
  • fastRewind()
