    Known drawbacks:
  • Requires the width and height defined explicitly. I.e. no "auto fit" mode. No 100% mode.
  • Does not fit the size of the wrapped element precisely. Requires different style for IE and FF browsers for correction
  • Incompatible with xhtml mode (it is EXT JS disease in general). Might cause losing the flexibility in Ajax mode.
