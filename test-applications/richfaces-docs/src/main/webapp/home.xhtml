RichFaces Documentation Application

RichFaces documentation team from now on (3.3.1 version of RichFaces) will be using RDA while documenting RichFaces components. The main idea of the application is to make sure that all code snippets we provide in the Guide are working in a real application. Besides, it will help us keep the examples in one place so that any doc team member can continue what the other has started.

Developed and maintained by Michael Sorokin. Feel free to email me: msorokin@exadel.com
